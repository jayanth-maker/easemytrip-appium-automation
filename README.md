# EaseMyTrip Mobile Automation Framework

This project is a mobile automation framework for the EaseMyTrip Android application built using Python and Appium.

## Project Overview

The framework automates the flight search functionality of the EaseMyTrip mobile application.

It demonstrates:

• Appium based mobile automation  
• Page Object Model (POM) design pattern  
• AI generated test cases  
• Screenshot capture for failures  
• HTML test reports  
• Config driven framework  
• Git version control

## Project Structure

EasyTripAutomation/
│
├── ai/                     # AI generated test case module
├── config/                 # Configuration file
├── pages/                  # Page Object Model classes
├── reports/                # HTML test reports
├── screenshots/            # Failure screenshots
├── test_data/              # AI generated test data
├── tests/                  # Test scripts
├── utils/                  # Driver setup & report generator
│
├── test_connection.py      # Basic driver test
├── easemytrip_test.py      # Initial automation script
└── README.md               # Project documentation

## Prerequisites

Before running the framework install:

• Python 3.9+
• Appium Server
• Android Studio
• Android Emulator or Real Device
• Node.js
